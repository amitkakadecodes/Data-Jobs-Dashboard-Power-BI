# Data-Jobs-Dashboard-Power-BI

This dashboard was created for someone who is looking to **trascition into data job, looking for a data job or swapping their respective data job**, to solve common problem: Imformation about data job market. Using real-world dataset of 2024 data science job posting including job title, salary and location. This project provides easy to use interface to explore data job market.

# Skills

**Data transformation (ETL) with power query:**
Cleaned, shaped and prepared raw data for data analysis

**Measures:**
Formulated measures and KPIs like 'Median yearly salary' and 'job count'
